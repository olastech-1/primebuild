# PrimeBuild Construction

A modern, fully responsive single-page website for a construction company.  
Built to showcase services, completed projects, and generate client inquiries.

## 🚀 Live Demo
[View Website](https://primebuild-rho.vercel.app)

## ✨ Key Features
- **Clinical Mobile Responsiveness** - Perfect on Mobile, Tablet, Desktop
- **Modern UI** - Clean design with Tailwind CSS + Gradient Hero
- **Single Page Layout** - Smooth scroll: Home, About, Services, Projects, Contact
- **Company Stats** - 150+ Projects Completed, 10+ Years Experience
- **4 Core Services** - Residential, Commercial, Renovation, Architecture
- **Projects Gallery** - Showcasing featured construction work
- **Professional Footer** - 4-column layout with contact info
- **Fast & Lightweight** - No build tools. Pure HTML + Tailwind CDN

## 🛠️ Tech Stack
- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS via CDN
- **Font Awesome 6.4.2** - Icons
- **Unsplash** - High quality images
